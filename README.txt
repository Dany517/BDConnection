Paso 1: Configurar Git en Visual Studio

Abrir Visual Studio 2022:
Inicia Visual Studio 2022 en tu equipo.

Acceder a la configuración de Git:
Ve a "Herramientas" > "Opciones" en el menú superior.
En la ventana de opciones, navega hasta "Control de Código Fuente" > "Git Global Settings".

Configurar tu cuenta de Git:
Si nunca has usado Git, agrega tu nombre y correo electrónico.

Paso 2: Clonar el repositorio

Abrir la opción de clonar:
En la página de inicio de Visual Studio, haz clic en "Clonar un repositorio".

Pegar la URL del repositorio:
Copia la URL del repositorio desde tu proveedor (por ejemplo, GitHub).
Pégala en el campo "Repositorio para clonar".

Seleccionar la ruta de tu proyecto:
Elige un directorio local donde deseas guardar el repositorio clonado.

Clonar el repositorio:
Haz clic en "Clonar". Visual Studio descargará el contenido del repositorio.
